Easily add a store banner to your site

== Description ==

Add a banner showing the content of your Society6 store to your wordpress blog.

NOTE! The widget is based on the store promotion code available from Society6 and hence the widget calls to the script hosted with them. 
To see the original code visit: http://society6.com/account/banner

== Installation ==

1. Upload the entire `dbhn-society6-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets and add the Society6 Widget to the sidebar of your choice
4. Add your info and enjoy!
